# Database Lab Task

Below is the SQL code for creating the database and the employee table.

```sql
-- 1. Database banayein
CREATE DATABASE LabTaskDB;
USE LabTaskDB;

-- 2. Table banayein (10 features ke saath)
CREATE TABLE Employees (
    emp_id INT PRIMARY KEY,
    full_name VARCHAR(100),
    age INT,
    department VARCHAR(50),
    salary DECIMAL(10, 2),
    joining_date DATE,
    city VARCHAR(50),
    email VARCHAR(100),
    performance_rating INT,
    work_status VARCHAR(20)
);
